# 🛒 NovaCart

> A modern full-stack e-commerce platform built with **React**, **Vite**, **Tailwind CSS**, and **Supabase**.

NovaCart is a production-inspired e-commerce application that delivers a complete online shopping experience with secure authentication, category-based browsing, product management, wishlist, cart, checkout, delivery location, and a powerful admin dashboard.

# 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/Swapnil05-03/Novacart.git
```

### Install Dependencies

```bash
npm install
```

### Create Environment File

Create a `.env` file in the project root.

```env
VITE_SUPABASE_URL=YOUR_SUPABASE_URL
VITE_SUPABASE_ANON_KEY=YOUR_SUPABASE_ANON_KEY
```

### Start Development Server

```bash
npm run dev
```
